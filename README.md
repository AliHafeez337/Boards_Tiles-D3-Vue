# Boards - Tiles - D3

- [**Live demo**](https://board-ac1a6.web.app/)


## Project setup

#### Node.js

##### open terminal in the root directory (where 'package.json' file is present)
```
npm install
```

##### now in the same directory, to run the project...
```
npm start
```

#### Vue.js

##### move to the front-end directory and run
```
npm install
```

##### now in the same directory, to run the project...
```
npm run serve
```
